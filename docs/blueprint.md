# **App Name**: Password Puzzle

## Core Features:

- Puzzle Presentation: Display different puzzle types including math riddles, pattern recognition, basic coding clues, and cipher decoding.
- Hint System: Provide contextual hints that can be used to solve puzzles.
- Password Revelation: Reveal parts of the password upon successfully completing puzzles.
- Real-Time Scoreboard: Display individual scores in real-time.
- Admin Puzzle Generation: Generate puzzles with increasing difficulty for teams to solve as they progress through each level. AI tool may determine if difficulty should ramp up or down based on how users are performing at a certain level

## Style Guidelines:

- Primary color: Deep purple (#673AB7) for a mysterious and intellectual atmosphere.
- Background color: Light gray (#F5F5F5), close to white, for a clean, uncluttered backdrop that helps focus the user's attention.
- Accent color: Teal (#009688) to highlight clues and interactive elements, offering a stark visual contrast.
- Headline font: 'Space Grotesk' sans-serif for headlines and short descriptions, 'Inter' sans-serif for longer descriptions.
- Use abstract, minimalist icons to represent different puzzle types (e.g., math, coding, ciphers).
- A clean, card-based layout to present puzzles and hints in an organized manner.
- Subtle animations when puzzle elements are solved and new password parts are revealed.
